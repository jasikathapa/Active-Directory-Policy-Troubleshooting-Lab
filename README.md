# 🎫 IT Helpdesk Ticketing Simulation

This project simulates a real-world IT Helpdesk environment using ITIL-style incident management workflows. It demonstrates practical IT Support skills including user support, troubleshooting, Active Directory administration, and ticket documentation.

---

## 🧠 Skills Demonstrated

- 🎫 Incident ticket handling
- 👥 Active Directory user support
- 🔐 Password reset & account unlock procedures
- 🖥️ Windows troubleshooting
- 🌐 Network & domain connectivity troubleshooting
- 📁 Shared folder permissions management
- 📧 Email support scenarios

---

## 🎯 Objective

The purpose of this lab is to simulate common Helpdesk tasks and document the troubleshooting and resolution process used in enterprise IT environments.

---

## 📁 Ticket Scenarios

- 🔐 Password Reset Request
- 🚫 Account Lockout Resolution
- 📁 Shared Folder Access Issue
- 🌐 Domain Login Failure
- 📧 Outlook / Email Access Issue

---

## 🛠️ Environment

- Windows Server (AD DS)
- Windows 10/11 Clients
- Active Directory Users & Computers
- VirtualBox Lab Environment

---

## 📌 ITIL Concepts Simulated

- Incident Logging
- Prioritisation
- Troubleshooting
- Resolution Documentation
- User Communication
